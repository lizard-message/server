# 通讯协议

都为小端

## 客户端通信

    要做的功能:
    1. 握手
    2. 维持心跳
    3. 客户端订阅消息
    4. 服务器推送信息
    5. 客户端推送消息
    6. 
    . 提供应答

协议类型:

    服务器信息 => 1
    客户端信息 => 2
    ping => 3
    pong => 4
    推信息开始或继续 => 5
    推信息结束 => 6
    拉信息开始或继续 => 7
    拉消息结束 => 8
    应答 => 9

1. 握手

首先, 由服务器提供服务信息

    |2字节|2字节|4字节|2字节|4字节|
    |类型|版本|支持的服务的位掩码|心跳时间|可接收内容的最大长度|

位掩码:
    
    服务端支持推消息 => 1
    服务端支持拉消息 => 2


其次, 由客户端提供信息

    |2字节|2字节|1字节|1字节|
    |类型|版本|支持的服务的位掩码|是否支持ssl|
